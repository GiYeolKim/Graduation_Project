Flask

Flask is a lightweight Python web framework that simplifies web application development. It provides routing and response handling, supports database integration and form handling, and emphasizes simplicity. With Flask, you can quickly build dynamic web applications with minimal overhead.

Instant Nerf

This implementation provides four neural graphics primitives: neural radiance fields (NeRF), signed distance functions (SDFs), neural images, and neural volumes. It uses a multilayer perceptron (MLP) with multiresolution hash input encoding and the tiny-cuda-nn framework for fast training and rendering. With this implementation, you can train and render complex scenes, such as a fox or a factory robot, in a matter of seconds.

SFM

SFM is a computer vision technique used to reconstruct 3D structure from a set of images or videos. It involves estimating camera poses and 3D point locations by analyzing the motion and visual information captured in the input data. SFM is commonly used in applications such as 3D modeling, augmented reality, and robotics for understanding the geometry and spatial relationships of a scene.
